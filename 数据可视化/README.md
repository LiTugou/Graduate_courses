## 数据清洗
### 缺失值插补 
回归与随机回归插补的问题：
1) 不能使用含缺失值的变量作为预测变量
2) 插补的值可能越出已观测值范围（回归中以与预测值最接近的观测值插补，随机回归中截断）
#### 回归插补
以需要插补的变为目标，其他相关变量为预测变量建立模型，以预测值为插补值。  
注意事项：
1) 需要对每一变量独立进行插补（不能使用有缺失的样本观测信息）
